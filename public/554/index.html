BİL554 İnternet Uygulama Geliştirme Projesi

Proje adı: Prestamé

Proje takımı: Ali Onur Baykal

Açıklama: Prestamé, sosyal ağlarla iç içe kurulmuş bir borç takip sistemidir.

Yazılımın yapacağı işler (en az 5):

  1. Borç Takibi (Borç Eklemek, Kapatmak, Silmek, Düzenlemek)
  2. Sosyal Ağ Entegrasyonu (Sosyal Ağ İle Giriş, Sosyal Ağlarda Paylaşma)
  3. Grafik Üretimi (Sistem geneli ve kullanıcı için istatistiksel grafik üretimi)
  4. E-Mail ile uyarı (Vadesi gelen, yeni eklenen, ödenen borç takibi)
  5. Borç Listeleme (Ödenmiş ve Ödenmemişleri listeleme, yeni borç eklerken ödenmemiş borç sahipleri arasında arama)

Kullanılan teknoloji ve bileşenler:

  1. Ruby on Rails
  2. Twitter Bootstrap
  3. Twitter API
  4. Facebook Developer API
  5. GitHub Bash
  6. Heroku
  7. SQLite
  8. PostgreSQL
